# portal-cube
Simple Spinning-Cube program using GLFW and GLAD

## Requirements

- [CMake](https://cmake.org)
- [Ninja](https://ninja-build.org/)

## Building

```bash
git clone https://github.com/lovkiy164/portal-cube.git
cd portal-cube
mkdir build
cd build
cmake -G "Ninja" ..
ninja
./PortalCube
```
